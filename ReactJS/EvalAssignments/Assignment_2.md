# ReactJS Session 2 - Assignment

## Tasks:

**Task 1: Time Travel Tic-Tac-Toe** Extend the tic-tac-toe game with complete move history. Players should be able to jump back to any previous move and continue playing from that point. Use useState for game state and useEffect for managing game flow.

**Task 2: ASP.NET Core API Integration** Create React applications that consume all the Web APIs you built during ASP.NET Core Web API training. Implement full CRUD operations with proper error handling, loading states, and form validation for each API endpoint you previously created.
